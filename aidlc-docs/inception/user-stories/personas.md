# 테이블오더 서비스 — 사용자 페르소나

---

## Persona 1: 고객 (Customer)

**이름**: 김민수 (대표 고객)
**역할**: 매장 방문 고객 (테이블 태블릿 사용자)

**목표**:
- 대기 시간 없이 빠르게 메뉴를 탐색하고 주문하기
- 주문 상태를 실시간으로 확인하기
- 추가 주문을 편리하게 하기

**동기**:
- 직원을 부르지 않고 편하게 주문하고 싶음
- 메뉴 사진과 설명을 보고 신중하게 선택하고 싶음
- 현재까지 주문한 내역과 총 금액을 확인하고 싶음

**불편사항 (Pain Points)**:
- 복잡한 UI로 인해 주문에 시간이 오래 걸림
- 주문이 제대로 들어갔는지 확인이 안 됨
- 장바구니에 담은 메뉴가 사라짐 (새로고침 시)

**기술 수준**: 다양함 (스마트폰 사용 가능한 일반인 수준)
**사용 환경**: 매장 내 테이블에 고정된 태블릿, 터치 인터페이스

---

## Persona 2: 매장 관리자 (Admin)

**이름**: 박서연 (대표 관리자)
**역할**: 매장 운영자 / 점주

**목표**:
- 들어오는 주문을 실시간으로 확인하고 처리하기
- 메뉴와 카테고리를 효율적으로 관리하기
- 테이블별 주문 현황을 한눈에 파악하기
- 테이블 세션(고객 이용)을 관리하기

**동기**:
- 인건비를 절감하고 운영 효율성을 높이고 싶음
- 주문 누락 없이 정확하게 처리하고 싶음
- 매장 상황을 실시간으로 모니터링하고 싶음

**불편사항 (Pain Points)**:
- 바쁜 시간대에 주문이 밀려서 놓치는 경우
- 테이블별 주문 현황 파악이 어려움
- 메뉴 변경(가격, 품절 등)을 즉시 반영하기 어려움
- 이전 고객의 주문 내역이 새 고객에게 보이는 문제

**기술 수준**: 기본적인 웹 사용 가능 (전문 IT 지식 불필요)
**사용 환경**: PC 또는 태블릿 브라우저, 매장 내 Wi-Fi 환경
